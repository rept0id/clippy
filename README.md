# Usage

## How to start

Once you download one of the releases, simply run the executable.
Else, you can download source code, run `npm i` and then `npm run`.

# Build

There are build scripts located on package.json.
For Linux :
`npm run build_linux`
For Windows :
`npm run build_win`
`npm run build_win_portable`
